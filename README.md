# Roadmaps de Estadística

Árboles de decisión para elegir el test estadístico correcto. Material de apoyo del curso de nivelación en estadística (ITBA).

🌐 **Sitio online:** https://emanuelrodriguezbedeman.github.io/roadmaps-estadistica/

## Convenciones de color

- 🟧 **Naranja claro** — decisiones (chequear supuesto, contar grupos, etc.).
- 🟦 **Azul / 🟩 verde** — tests paramétricos (asumen normalidad).
- 🟧 **Naranja oscuro** — correcciones (Welch, Greenhouse-Geisser, Huynh-Feldt).
- 🟪 **Violeta** — alternativas no paramétricas (Wilcoxon, Mann-Whitney, Kruskal-Wallis).
