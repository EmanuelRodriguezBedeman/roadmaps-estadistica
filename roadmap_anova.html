ANOVA Factorial

2 factores: A (k₁ niv.) · B (k₂ niv.)
3 H₀ en simultáneo
H₀⁽¹⁾ · Efecto principal A
F_A = MS_A / MSE gl = (k₁−1, N−k₁·k₂)
H₀⁽²⁾ · Efecto principal B
F_B = MS_B / MSE gl = (k₂−1, N−k₁·k₂)
H₀⁽³⁾ · Interacción A×B
F_AB = MS_AB / MSE gl = ((k₁−1)(k₂−1), N−k₁·k₂)
⚠ Orden de interpretación
¿Interacción significativa?
SÍ
Analizar efectos simples NO interpretar A y B aislados
NO
Interpretar efectos principales por separado
Requisitos: mismos supuestos del 1F + tamaños iguales por celda. Si hay desbalanceo severo, usar GLM o suma de cuadrados tipo III.

Cálculos ANOVA de un factor

Suma de cuadrados Fórmula Grados de libertad
SSB (entre grupos) Σ nᵢ (X̄ᵢ − X̄)² k − 1
SSW (dentro / error) Σᵢ Σⱼ (Xᵢⱼ − X̄ᵢ)² N − k
SST (total) SSB + SSW N − 1
MST (intervarianza) SSB / (k − 1) —
MSE (intravarianza) SSW / (N − k) —
F-ratio MST / MSE (k−1, N−k)
η² (tamaño efecto) SSB / SST —

Checklist global ANOVA

  1. Explorar los datos — boxplot por grupo, QQ-plot, medias.
  2. Verificar supuestos — independencia (diseño), normalidad de residuos (Shapiro/Lilliefors), homocedasticidad (Levene), esfericidad (Mauchly, si es repetidas).
  3. Elegir test — 1F · 2F · medidas repetidas · Welch heterocedástico · Kruskal-Wallis / Friedman si falla normalidad.
  4. Calcular F-ratio y reportar p-valor + η² (proporción de varianza explicada).
  5. Si rechazo H₀ → post-hoc: Bonferroni (pocas, planeadas), Tukey HSD (todas las parejas), LSD Fisher (solo exploratorio).
  6. Reportar — F(gl₁, gl₂) = valor, p, η², qué test se usó y por qué.

Glosario rápido

  • Factor / tratamiento — variable independiente categórica.
  • Niveles — valores que toma el factor (ej.: A, B, C).
  • SSB (between) — variabilidad entre grupos = explicada por el factor.
  • SSW (within) — variabilidad dentro de los grupos = error / no explicada.
  • F-ratio = MST / MSE — si ≈ 1, no hay efecto del factor; si ≫ 1, sí.
  • η² = SSB / SST — tamaño del efecto.
  • TEF (family-wise error rate) — 1 − (1 − α)ʰ. Sin corrección, crece rápido con h.
  • Esfericidad — igualdad de varianzas de las diferencias entre pares de niveles (solo medidas repetidas).
