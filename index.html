¿Cómo leer cada roadmap?

  • Cajas naranjas son decisiones (chequear un supuesto, contar grupos, etc.).
  • Cajas azules / verdes son tests paramétricos (asumen normalidad).
  • Cajas naranjas oscuras son correcciones (Welch, Greenhouse-Geisser, Huynh-Feldt).
  • Cajas violetas son alternativas no paramétricas (Wilcoxon, Mann-Whitney, Kruskal-Wallis).

Las versiones .md son los diagramas en formato Mermaid — se pueden copiar y pegar en mermaid.live, GitHub, Notion o VS Code para renderizarlos interactivamente.
