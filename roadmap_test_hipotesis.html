Roadmap: ¿Qué test de hipótesis usar?

Árbol de decisión basado en el material del curso

¿Qué parámetro deseas evaluar?

Media (μ)

¿Cuántas muestras?
A. Una muestra — H₀: μ = μ₀
¿Conocés σ poblacional?
SÍ σ conocido
Test Z (x̄ − μ₀)/(σ/√n) ~ N(0,1)
NO σ desconocido
¿n grande (≥ 40)?
SÍ n grande → usar s en lugar de σ
Test Z asintótico (TCL)
NO n chico → evaluar normalidad
Shapiro-Wilk + QQ-plot
Normal
Test t de Student t.test(x, mu=μ₀)
No normal
Wilcoxon (1 muestra) wilcox.test(x, mu=μ₀)
Alternativa: transformar con Box-Cox y repetir el t-test.
B. Dos muestras independientes — H₀: μ₁ = μ₂
¿n₁ y n₂ grandes (≥ 40)?
SÍ
Test Z para 2 medias (asintótico)
NO → Shapiro en cada grupo
Alguno no normal
Mann-Whitney-Wilcoxon wilcox.test(x1, x2)
Ambos normales → Test F de Fisher
¿σ₁² = σ₂²?
Iguales
t pooled (varianza combinada) t.test(x1, x2, var.equal=TRUE)
Distintas
Test t de Welch t.test(x1, x2)
C. Muestras apareadas — H₀: μ_D = 0
PASO 1 Calcular D = X₁ − X₂
PASO 2 Shapiro-Wilk sobre D
Normal
Test t pareado t.test(x1, x2, paired=TRUE)
No normal
Wilcoxon signed-rank wilcox.test(x1, x2, paired=TRUE)

Proporción (p)

¿Cuántas proporciones?
Una proporción — H₀: p = p₀
Verificar éxito-fracaso:
n·p₀ ≥ 10 y n·(1−p₀) ≥ 10
SÍ muestra grande
Test Z para proporción prop.test(x, n, p=p₀)
NO muestra chica
Test exacto binomial binom.test(x, n, p=p₀)
Dos proporciones — H₀: p₁ = p₂
Muestras grandes
Test Z para 2 proporciones prop.test(c(x1,x2), c(n1,n2))
Usa p̂ combinada bajo H₀:
p̂ = (x₁+x₂)/(n₁+n₂)

Varianza (σ²)

Comparar varianzas de 2 grupos
H₀: σ₁² = σ₂²
¿Los datos son normales?
SÍ normales
Test F de Fisher var.test(x1, x2)
NO normales
Test de Levene
Test de Fligner-Killeen
Contexto: este test suele ser un paso intermedio dentro del flujo de comparación de medias (A.3), no un fin en sí mismo. Determina si usar t pooled o t de Welch.

Mediana

Tests no paramétricos directos
(sin asumir normalidad)
Una muestra vs valor m₀
Wilcoxon 1 muestra wilcox.test(x, mu=m₀)
Dos muestras independientes
Mann-Whitney-Wilcoxon wilcox.test(x1, x2)
Muestras apareadas
Wilcoxon signed-rank wilcox.test(x1, x2, paired=TRUE)
Cuándo usarlo:
  • hay outliers
  • n chico
  • la normalidad se rechaza
  • no querés transformar

Checklist previo a cualquier test

  1. Explorar los datos — histograma, boxplot, QQ-plot.
  2. Verificar supuestos — normalidad (Shapiro-Wilk), independencia, igualdad de varianzas cuando corresponda.
  3. Elegir el test siguiendo este roadmap según el parámetro y el diseño.
  4. Decidir con el p-valor y reportar tamaño del efecto (d de Cohen). La significancia estadística no implica relevancia práctica.
  5. Complementar con intervalo de confianza siempre que sea posible — aporta más información que el p-valor solo.
