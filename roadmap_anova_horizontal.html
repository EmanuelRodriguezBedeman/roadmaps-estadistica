⚡ Decisión inicial — ubicar el problema

2 grupos → test t (otro roadmap)
k ≥ 3 grupos → ANOVA
1 factor → ANOVA 1F
2+ factores → ANOVA factorial
Mismos sujetos → ANOVA medidas repetidas
ANOVA supone independencia, normalidad de residuos y homocedasticidad. Si falla normalidad de forma extrema con n chico → Kruskal-Wallis. Si falla solo homocedasticidad → ANOVA de Welch. Si es medidas repetidas se agrega esfericidad (Mauchly) → correcciones GG / HF.
